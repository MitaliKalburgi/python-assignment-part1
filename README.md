# python-assignment-part1

## Student Grade Tracker — Python Assignment (Part 1)

## 📌 Description
This project is a command-line based Student Grade Tracker built using core Python concepts.  
It demonstrates data cleaning, loops, conditionals, and string manipulation.

---

## 📂 Tasks Implemented

### 🔹 Task 1 — Data Parsing & Profile Cleaning
- Cleaned student names (trim + title case)
- Converted roll numbers to integers
- Converted marks from string to list
- Validated names
- Printed formatted student profiles

---

### 🔹 Task 2 — Marks Analysis
- Displayed subjects with marks and grades using a **for loop**
- Calculated:
  - Total marks
  - Average marks
  - Highest & lowest scoring subjects
- Implemented a **while loop** for dynamic subject entry
- Handled invalid inputs and duplicate subjects

---

### 🔹 Task 3 — Class Performance Summary
- Calculated average marks for each student
- Assigned Pass/Fail status
- Generated formatted class report
- Computed:
  - Number of students passed/failed
  - Class topper
  - Class average

---

### 🔹 Task 4 — String Manipulation Utility
- Cleaned essay text using `.strip()`
- Converted text to Title Case
- Counted occurrences of "python"
- Replaced text with formatted output
- Split essay into sentences
- Printed numbered sentences

---
